# 🚀 Local Development

Right-click:

```
index.html
```

Choose:

```
Open with Live Server
```

---

# 🚀 Deploy to GitHub Pages

1. Create a GitHub repository.

2. Upload all files.

3. Commit your changes.

4. Open:

```
Settings
```

↓

```
Pages
```

↓

```
Deploy from Branch
```

↓

Branch

```
main
```

Folder

```
/root
```

Click **Save**.

After about one minute your website will be live.
